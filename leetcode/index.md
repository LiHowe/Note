# 常识



1. 时间复杂度 T(n)

   * 时间频度: 一个算法中语句执行次数, 记为T(n), n为问题的规模

   * 时间复杂度: 

     常见的算法时间复杂度由小到大依次为: 

     **Ο(1)＜Ο(log<sub>2</sub>n\)＜Ο(n)＜Ο(nlog<sub>2</sub>n)＜Ο(n<sup>2</sup>)＜Ο(n<sup>3</sup>)＜…＜Ο(2<sup>n</sup>)＜Ο(n!)**

     ... 前面的称为多项式时间, 后面称为指数时间

     * 常数阶 **Ο(1)**
     * 对数阶 **Ο(log<sub>2</sub>n\)**
     * 线性阶 **Ο(n)**
     * 线性对数阶 **Ο(nlog<sub>2</sub>n)**
     * 平方阶 **Ο(n<sup>2</sup>)**
     * 立方阶 **Ο(n<sup>3</sup>)**
     * K次方阶 **Ο(n<sup>k</sup>)**
     * 指数阶 **Ο(2<sup>n</sup>)**
